#mark2web  
意如其名， [mark2web][1] 用于由MarkDown文件生成Web页面。 准确的说是由MarkDown组成的文件系统生成单页面应用（Single Page Application），效果参考 [Backbone.js][2]，» 项目线上 [demo][3]。

出发点:

- 由 [MarkDown][4] 写作；
- 用于技术文档或产品手册写作及阅读；
- 使用 [Github Pages][5] 部署（当然也可以部署到自己的主机上），对外展示。

如果你正有同样的想法，请参考[如何使用mark2web][6]。
